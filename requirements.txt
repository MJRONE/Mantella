# Unpinned requirements for MantellaMJR (new version).
# A copy of the original pinned list (as shipped with the new version) is kept
# in `requirements.pinned.txt` for reference.
#
# Rationale for removing pins:
#   * The original pins target Python 3.11 and do not have wheels on Python 3.13.
#   * Most of the packages below are already installed from OldMantella with
#     newer versions that work fine for both Mantella instances.
#   * Without pins, `pip install -r requirements.txt` leaves already-installed
#     packages untouched (its default upgrade strategy is `only-if-needed`),
#     so this file won't downgrade your OldMantella environment.
#
# If you ever end up in a fresh environment, feel free to reintroduce the pins
# from `requirements.pinned.txt`.

setuptools
pandas
tiktoken
httpx
openai
aiohttp
faster-whisper
requests
pyaudio
soundfile
numpy
charset-normalizer
fastapi
huggingface_hub
gradio
pywin32; platform_system == "Windows"
mss
opencv-python-headless
sounddevice
regex
silero-vad-lite
# playsound 1.3.0 has a broken setup.py (OSError: could not get source code) on
# Python 3.13, so pin to 1.2.2 (last pure-Python release). Only used as a
# fallback when winsound is unavailable, so this can also be safely omitted
# on Windows if it causes trouble.
playsound==1.2.2
# Moonshine (optional speech-to-text). Disabled because its setup.py relies on
# pkg_resources, which has been dropped from recent setuptools, and because it
# is not required when using Whisper as the STT service. The src code already
# imports it inside a try/except ModuleNotFoundError, so leaving it uninstalled
# is safe.
# git+https://github.com/usefulsensors/moonshine.git@2f6282347950c8d711bd7319287babb87ec5a92d#subdirectory=moonshine-onnx
